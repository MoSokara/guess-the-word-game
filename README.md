# 🎮 Guess The Word Game

A modern and interactive **Word Guessing Game** inspired by Wordle, built using pure **HTML, CSS, and JavaScript** with clean architecture and dynamic UI.

---

## 🚀 Live Demo

https://mosokara.github.io/guess-the-word-game/

---

## 📌 Overview

Guess The Word is a browser-based game where players try to guess a hidden word within a limited number of attempts.  
The game provides real-time feedback, hints, and customizable difficulty levels.

---

## ✨ Features

- 🎯 Multiple difficulty levels (Easy, Medium, Hard)
- 🔢 Adjustable word length (4, 5, 6 letters)
- 🔁 Custom number of tries
- 💡 Smart hint system
- 🎨 Dynamic UI with visual feedback
- ⌨️ Keyboard navigation support
- 📱 Responsive design
- 🔄 Restart game functionality
- 📊 Win/Lose popups with stats

---

## 🎮 How to Play

1. Choose your settings (difficulty, word length, tries)
2. Start the game
3. Enter letters to guess the word
4. Use hints if needed
5. Submit your guess

### Feedback Colors:

- 🟩 Correct letter in correct position
- 🟨 Correct letter in wrong position
- ⬛ Letter not in the word

---

## 🧠 Game Logic

- Random word is generated based on difficulty and length
- Each guess is validated and compared letter-by-letter
- Inputs are dynamically generated based on settings
- Game state is managed using JavaScript

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & UI
- **JavaScript (ES6+)** – Logic & Interactivity
